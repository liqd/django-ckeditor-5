# CKEditor Image Aspect Ratio Styles

## Overview

CKEditor supports two image aspect ratio options via CSS classes. These CSS rules must be defined in your project's stylesheets.

## CSS Classes

### 4:3 Aspect Ratio
- **Class:** `.ck-content figure.image-aspect-ratio-43`
- Applied when the 4:3 button is clicked

### 2:1 Aspect Ratio
- **Class:** `.ck-content figure.image-aspect-ratio-21`
- Applied when the 2:1 button is clicked

## Additional Image Upload Buttons

### 4:3 Upload
- **Toolbar item:** `imageUpload43`
- Uploads image and applies 4:3 aspect ratio

### 2:1 Upload
- **Toolbar item:** `imageUpload21`
- Uploads image and applies 2:1 aspect ratio
